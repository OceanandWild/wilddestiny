## 🎮 Cómo Usar

### 1️⃣ Navegar
```
Toca ☰ (arriba derecha) → Abre menú
```

### 2️⃣ Equipar Personaje
```
☰ → Characters → Selecciona → Equipar
```

### 3️⃣ Jugar Raid
```
☰ → Raids → Juega con controles táctiles
```

## 🎮 Controles

### En Menú
- **Toca ☰**: Abrir menú
- **Toca opciones**: Navegar
- **Toca habilidades**: Ver info

### En Juego
- **Toca izquierda**: Joystick (mover)
- **Toca derecha**: Botones (habilidades)
- **Arrastra joystick**: Controlar dirección
