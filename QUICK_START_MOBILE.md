# 🚀 Inicio Rápido - Controles Móviles

## 📱 ¿Cómo Jugar en Móvil?

### 1️⃣ Abre el Juego
- Abre `index.html` en tu navegador móvil
- El sistema detectará automáticamente que estás en móvil
- La pantalla rotará a horizontal automáticamente

### 2️⃣ Equipar Personaje/Clase
- Usa el menú táctil para seleccionar tu personaje
- Toca el botón de hamburguesa (☰) para abrir el menú
- Selecciona "Characters" o "Classes"
- Toca el personaje/clase que quieras usar

### 3️⃣ Iniciar Raid
- Abre el menú (☰)
- Toca "Raids" o "Modo Práctica"
- ¡El juego comenzará!

### 4️⃣ Controles Básicos

#### 🕹️ Movimiento
```
┌─────────────────┐
│                 │
│  👆 Toca aquí   │
│  (Izquierda)    │
│                 │
│  Aparecerá un   │
│  joystick       │
│                 │
│  Arrastra para  │
│  moverte        │
└─────────────────┘
```

#### ⚔️ Habilidades
```
┌─────────────────────────────────┐
│  Toca los botones en la parte   │
│  inferior de la pantalla         │
│                                  │
│  [Q] [W] [E] [R] [T] [Y] [U] [I]│
```

## 🎮 Controles Detallados

### Durante Raids

| Acción | Control |
|--------|---------|
| **Mover** | Toca y arrastra en la mitad izquierda |
| **Habilidad 1** | Toca botón Q |
| **Habilidad 2** | Toca botón W |
| **Habilidad 3** | Toca botón E |
| **Habilidad 4** | Toca botón R |
| **Semi-Ultimate** | Toca botón T |
| **Ultimate** | Toca botón Y |
| **Pausar** | Usa el menú |

### En Modo Práctica

| Acción | Control |
|--------|---------|
| **Mover** | Joystick (izquierda) |
| **Habilidades** | Botones inferiores |
| **Panel Control** | Toca panel (derecha) |
| **Agregar Dummy** | Toca botón en panel |
| **Reset** | Toca botón en panel |
| **Salir** | Toca botón en panel |

---

## 💡 Consejos Rápidos

### ✅ Haz Esto
- ✅ Usa ambas manos (izquierda para mover, derecha para habilidades)
- ✅ Mantén el dedo en el joystick para movimiento continuo
- ✅ Toca rápidamente las habilidades para combos
- ✅ Practica en Modo Práctica antes de raids difíciles
- ✅ Juega en orientación horizontal

### ❌ Evita Esto
- ❌ No toques con múltiples dedos en el joystick
- ❌ No intentes usar teclado (no funcionará)
- ❌ No juegues en modo vertical (la experiencia no es óptima)
- ❌ No toques fuera de las áreas designadas

## 🔧 Solución Rápida de Problemas

### ❓ El joystick no aparece
**Solución**: Toca la mitad izquierda de la pantalla durante una raid o práctica

### ❓ La pantalla no rota
**Solución**: Rota manualmente tu dispositivo a horizontal

### ❓ Las habilidades no funcionan
**Solución**: Asegúrate de tener una clase equipada y que no estén en cooldown

### ❓ El juego va lento
**Solución**: Cierra otras apps, reduce el brillo, o reinicia el navegador

## 🎓 Tutorial Paso a Paso

### Primera Vez Jugando en Móvil

#### Paso 1: Configuración Inicial
1. Abre el juego en tu navegador móvil
2. Espera a que cargue completamente
3. Verás el mensaje "📱 Controles Móviles Activos"

#### Paso 2: Seleccionar Personaje
1. Toca el botón de menú (☰)
2. Toca "Characters" o "Classes"
3. Selecciona tu personaje favorito
4. Toca "Equipar"

#### Paso 3: Primera Raid
1. Abre el menú (☰)
2. Toca "Raids"
3. Espera a que inicie
4. Toca la izquierda para ver el joystick
5. Arrastra para moverte
6. Toca las habilidades para atacar

#### Paso 4: Dominar los Controles
1. Practica moverte en círculos
2. Prueba cada habilidad
3. Intenta combos simples
4. Esquiva enemigos
5. ¡Completa tu primera oleada!
